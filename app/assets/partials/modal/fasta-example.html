FASTA Specification and Example

FASTA files are the standard format for sequence data. Below is an illustrative example and an example with data. Any file format is fine as long as it is ultimately plain text (.txt, .fasta). Please limit to less than 20 MB for now.

>SAMPLE NAME (spaces, commas allowed but must start with a >)
NUCLEOTIDE SEQUENCE (A, C, T, G only no spaces)
CAN BE ON MANY LINES OF UNEQUAL LENGTH
